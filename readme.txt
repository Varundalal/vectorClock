Instructions to run the implementation of Vector Clocks in C++ language. 

1. The project only consists of a stand alone single vClock.cpp file.

USING IDE (Eclipse (MacOS/Linux/Windows) :

1. Create a new c++ project named vClock
2. Copy all the data from the .cpp file to the new main file.
3. Compile and Run.

USING Terminal (MacOS/Linux) :

1. Open terminal and navigate to the location of the file using cd command.
2. Once at the location of the cpp file type the following commands : 

   >>gcc vClock.cpp -o vClock.out  
   >>./vClock.out
